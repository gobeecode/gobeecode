
## Setup Instructions
1. Login to [OpenAI](https://platform.openai.com)
2. Add payment details [here](https://platform.openai.com/settings/organization/billing/overview) and add the minimum amount of 5$.
3. Generate API key [here](https://platform.openai.com/settings/organization/api-keys). Remember to save the key, it will be visible only at the time of generation.
